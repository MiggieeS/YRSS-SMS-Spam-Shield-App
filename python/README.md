
# SETUP PYTHON VIRTUAL ENVIRONMENT

Use ***Command Prompt*** not ***Windows PowerShell***.
1. `cd .\python`
2. `python -m venv venv`
3. `.\venv\Scripts\activate`
4. `python -m pip install -r requirements.txt`
5. `deactivate`
